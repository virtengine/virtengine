# Continuous Compliance Program

This program sustains SOC 2 Type II compliance beyond the initial audit.

## Status

- Program status: Active
- Effective date: 2026-02-06

## Program Elements

- Control owners and quarterly reviews
- Automated evidence collection with monthly cadence
- Policy and procedure annual review cycle
- Continuous monitoring and incident response exercises

## Governance

- Compliance steering meeting quarterly
- Risk register reviewed quarterly
- Annual control refresh aligned to audit period

## Metrics

- Evidence collection completion rate
- Access review completion rate
- Time to remediate control gaps
