# Gap Analysis and Remediation

Gap analysis identifies missing or weak controls and tracks remediation to closure.

## Method

- Compare current control implementation to objectives in control-objectives.md.
- Evaluate control effectiveness using evidence from the last three months.
- Record gaps, owners, and target completion dates.

## Remediation Tracking

- Use the compliance tracker to record status updates.
- Validate remediation with updated evidence and re testing.

## Reporting

- Monthly status summary to Security and Risk.
- Escalate critical gaps to leadership within five business days.
