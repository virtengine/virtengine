# Vendor Risk Management

This document defines the vendor risk management program for VirtEngine.

## Scope

Critical vendors include cloud infrastructure providers, security tooling, and data processors.

## Due Diligence

- Security questionnaire and evidence review before onboarding.
- Contract requirements for security, confidentiality, and incident notification.
- Verification of compliance reports when available.

## Ongoing Monitoring

- Annual re assessment for critical vendors.
- Continuous monitoring of security advisories and SLA performance.
- Offboarding controls for access and data disposal.

## Evidence

- Vendor assessment records and approvals.
- Contract summaries and compliance attestations.
