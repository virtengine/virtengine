## 2. Description of Services

VirtEngine provides a decentralized, blockchain-based cloud computing marketplace built on Cosmos SDK that connects:

- **Tenants**: Users who require computing resources (CPU, GPU, memory, storage)
- **Providers**: Operators who offer computing capacity to lease
- **Validators**: Blockchain node operators who secure the network

### 2.1 Core Services

**Blockchain Network**: A Cosmos SDK-based blockchain implementing a decentralized exchange for cloud computing resources, powered by CometBFT consensus.

**VEID (Verifiable Electronic Identity)**: An ML-powered identity verification system that enables:
- Biometric verification (facial recognition, liveness detection)
- Encrypted identity scope management
- Consent-based data sharing
- Provider-specific access controls

**Marketplace**: A decentralized platform for discovering, bidding on, and leasing computing resources with:
- Order creation and bid matching
- Lease agreements and escrow settlements
- Usage metering and billing
- Service level monitoring

**Provider Services**: Off-chain infrastructure adapters supporting:
- Kubernetes, OpenStack, AWS, Azure, VMware, Ansible
- Automated deployment and lifecycle management
- Resource usage tracking and reporting
